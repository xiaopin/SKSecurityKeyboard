# SKSecurityKeyboard

iOS 安全键盘

## 特性

- 支持多种数字键盘类型(纯数字、带小数点、身份证、随机等)
- 完美支持长按删除功能
- 支持按键音效果

## 用法

- 将 `SKSecurityKeyboard` 文件夹拖入项目
- 设置 UITextField/UITextView 的 `enabledSecurityKeyboard` 属性为 `YES` 即可
